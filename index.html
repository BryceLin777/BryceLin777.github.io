使用Typora的快捷键

在Typora中，有一些常用的快捷键可以帮助您更高效地编写Markdown文档。以下是一些常见的快捷键：

  1. 标题快捷键：
    • Ctrl + 1~6（Windows）：设置所选文本为一级到六级标题。
    • Ctrl + 0（Windows）：取消标题格式。
  2. 格式化文本快捷键：
    • Ctrl + B（Windows）：加粗所选文本或当前光标位置的单词。
    • Ctrl + I（Windows）：斜体化所选文本或当前光标位置的单词。
    • Ctrl + U（Windows）/ Cmd + U（macOS）：下划线所选文本或当前光标位置的单词。
    • Ctrl + Shift + ]（Windows）：增大所选文本的标题级别。
    • Ctrl + Shift + [（Windows）：减小所选文本的标题级别。
  3. 列表快捷键：
    • Ctrl + Shift + [数字]（Windows）：设置所选行或段落为有序列表（数字为列表项显示的数字）。
    • Ctrl + Shift + 8（Windows）：设置所选行或段落为无序列表。
  4. 引用快捷键：
    • Ctrl + Shift + >（Windows）：设置所选行或段落为引用块。
    • Ctrl + Shift + <（Windows）：减少所选行或段落的引用级别。
  5. 插入链接和图片快捷键：
    • Ctrl + K（Windows）：插入链接。
    • Ctrl + Shift + I（Windows）：插入图片。
  6. 其他常用快捷键：
    • Ctrl + Z（Windows）：撤销上一步操作。
    • Ctrl + Shift + Z（Windows）：重做上一步操作。
    • Ctrl + S（Windows）：保存文档。
    • Ctrl + F（Windows）：查找文本。
    • Ctrl + H（Windows）：替换文本。
